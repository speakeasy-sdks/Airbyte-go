# AttemptFailureType

Categorizes well known errors into types for programmatic handling. If not set, the type of error is not well known.


## Values

| Name                                   | Value                                  |
| -------------------------------------- | -------------------------------------- |
| `AttemptFailureTypeConfigError`        | config_error                           |
| `AttemptFailureTypeSystemError`        | system_error                           |
| `AttemptFailureTypeManualCancellation` | manual_cancellation                    |
| `AttemptFailureTypeRefreshSchema`      | refresh_schema                         |
